Hungarian Pronunciation Dictionary

This project involves creating a dictionary for Hungarian where each written word in Hungarian is assigned a phonemic form. This resource is to be used as an aid in linguistic and computational research, both specific to Hungarian as well as cross-linguistic.

Files:
  • Hungarian Pronunciation Dictionary -- columns are tab-deliminated.
  • IPA Chart of Hungarian consonants
  • IPA Chart of Hungarian vowels
  • .dx1 file (description) -- appropriate for use with John Goldsmith's phonological complexity calculator
  • Hungarian wordlist with CV structure (Kornai, Szigetvari)
The version of the pronunciation dictionary posted here has 14,529 entries. For another version containing 1.2 million entries, part-of-speech tags, and arguably more reliable frequency data, please contact me.
